## Projects

### [Zodiac & Writing Style Study](./zodiac-writing-study/)
**Can Sun Signs Influence Writing Style?**

A large-scale computational analysis examining whether zodiac signs are associated with language expression patterns.

- **Sample Size**: 17,872 authors from Project Gutenberg
- **Features**: 231 dimensions (Empath 194 + Style 37)
- **Methods**: Kruskal-Wallis test, Bonferroni correction, K-means clustering
- **Conclusion**: No statistically significant association found between sun signs and writing style
